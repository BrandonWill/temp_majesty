## Section Type Reference

### Graphics Pipeline Sections

| Section | Role | Entry Name Format | Notes |
|---------|------|-------------------|-------|
| `IMAG` | Animation metadata | `XXYY` + description (20 chars) | Defines frame count, dimensions, references TILE indices |
| `TILE` | Sprite pixel data | `[u32 index][description]` | RLE-encoded palette-indexed sprites |
| `SPLT` | Color palettes | Named (palette ID) | 768 bytes each (256×RGB). READ-ONLY! |
| `CUT`  | Cutscene frames | Named | Cutscene animation sequence data |
| `PICT` | Full-frame images | Named | Loading screens, cinematics (uncompressed bitmaps) |
| `PALT` | Palettes (UI) | Named | Alternative palette format in interfacedata |
| `FONT` | Bitmap fonts | Named | Font glyph data |

### Game Logic Sections

| Section | Role | Notes |
|---------|------|-------|
| `DUNT` | Unit type definitions | Stats, abilities, ImageIDBase, building refs |
| `DACT` | Action/spell definitions | Spell type, GPL function, timeout, effects |
| `DMOV` | Movement type defs | Speed, turning, animation parameters |
| `DTPT` | Terrain point types | Terrain tile properties |
| `DTRN` | Terrain config | Map terrain parameters |
| `DATA` | Generic data blobs | Miscellaneous binary configs |
| `BTDT` | Behavior trees | AI decision tree data (expansion only) |
| `DFCT` | Factory/company info | Developer identification |

### Audio Sections

| Section | Role | Notes |
|---------|------|-------|
| `WAVE` | Audio clips | WAV format sound effects and voice lines |
| `DSND` | Sound name mappings | Maps logical sound names to WAVE entries |
| `DSDP` | Sound presets | Playback parameters (volume, falloff, etc.) |
| `DSDG` | Sound groups | Groups of related sounds for random selection |

### Text/UI Sections

| Section | Role | Notes |
|---------|------|-------|
| `STRT` | String tables | Localized UI/game text |
| `SMNU` | Menu definitions | Menu layout and string data |
